HTML 태그 정의

HTML
HyperText Markup Language의 약자로, 웹 페이지를 작성하는 데 사용되는 표준 마크업 언어입니다.
CSS
Cascading Style Sheets의 약자로, HTML 요소의 스타일을 지정하는 데 사용되는 언어입니다.
JavaScript
웹 페이지에 동적인 기능을 추가하는 데 사용되는 프로그래밍 언어입니다.
DOM
Document Object Model의 약자로, HTML 문서의 구조를 프로그래밍적으로 조작할 수 있는 인터페이스입니다.
API
Application Programming Interface의 약자로, 소프트웨어 간의 상호작용을 정의하는 규칙입니다.
